About <ol> tag

<ol> is a special tags that help us list items or elements. In <ol> case, those items are listed with number, which is designed for ordered list.
This is the example for <ol> tag:

  1. This is the first item.
  2. This is the second item.
  3. This is the third item.

About <ul> tag

<ul> is also used to list items or elements. But when we use <ul> tag, we are meaning that this is an unordered list.
This is the example of <ul> tag:

  • This is an item.
  • This is another item.
  • This is the final item.

About hyperlink

Hyperlinks are a special element of the web, which makes it a "web". They are represented in HTML by <a>, which means anchor link. These are some types of anchor link, or reference:

  1. Absolute links/reference: usually point to another website. For example: https://en.wikipedia.org/wiki/Main_Page
  2. Relative links/reference: usually point to another file within the same folder that contain the file of the web. For example: BasicTag.html

We have a variety of tags that can help display text. They can be divided into two main groups: block and inline. Here are some useful tags from those groups:

Block

  • <div>: Defines a division or section in an HTML document. Used as a container for other elements.
    <div>This is a div block</div>
  • <p>: Represents a paragraph of text.
    <p>This is a paragraph.</p>
  • <header>: Represents introductory content, typically a group of introductory or navigational aids.
    <header>Header content</header>
  • <footer>: Represents a footer for its nearest sectioning content or root element.
    <footer>Footer content</footer>
  • <section>: Defines a section in a document.
    <section>Section content</section>
  • <article>: Represents a self-contained composition in a document.
    <article>Article content</article>
  • <nav>: Defines navigation links.
    <nav><a href="#">Home</a></nav>
  • <aside>: Represents content aside from the content it is placed in (like a sidebar).
    <aside>Sidebar content</aside>
  • <blockquote>: Indicates extended quotations.
    <blockquote>A famous quote.</blockquote>
  • <pre>: Displays preformatted text.
    <pre> Preformatted text </pre>
  • <hr>: Creates a thematic break (horizontal rule) in the page.
    <hr>
  • <address>: Defines contact information for the author/owner of a document or article.
    <address>123 Main St, City, Country</address>
  • <details>: Specifies additional details that the user can view or hide on demand.
    <details>\n <summary>More info</summary>\n This is hidden by default.</details>
  • <summary>: Defines a visible heading for a <details> element.
    <details>\n <summary>Click to expand</summary>\n Hidden content here.</details>

Inline

  • <span>: A generic inline container for phrasing content.
    <span style="color:blue;">This is a span</span>
  • <cite>: Represents the title of a work (book, website, etc.).
    <cite>HTML5 Specification</cite>
  • <abbr>: Represents an abbreviation or acronym, optionally with a full description.
    <abbr title="World Health Organization">WHO</abbr>
  • <time>: Represents a specific period in time.
    <time datetime="2025-08-09">August 9, 2025</time>
  • <code>: Displays a fragment of computer code.
    <code>console.log('Hello')</code>
  • <sub>: Subscript text.
    H<sub>2</sub>O
  • <sup>: Superscript text.
    x<sup>2</sup>

Tags that need more

  • <button>: Represents a clickable button.
    <button>Click me!</button>
  • <meter>: Represents a scalar measurement within a known range.
    <meter value="0.7">70%</meter>
  • <progress>: Displays the progress of a task.
    <progress value="50" max="100">50%</progress>
  • <iframe>: Embeds another HTML page within the current page.
    <iframe src="https://www.example.com" width="200" height="50"></iframe>
  • <bdo>: Overrides the current text direction.
    <bdo dir="rtl">Right to left</bdo> | <bdo dir="ltr">Left to right</bdo>
  • <map> and <area>: Define image maps and clickable areas on images.
    <img src="bangmaacsii.png" usemap="#asciimap">
    <map name="asciimap">
    <area shape="rect" coords="0,0,50,50" href="https://www.ascii-code.com/" alt="ASCII">
    </map>
